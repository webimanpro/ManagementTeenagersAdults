## 🚀 نصب و راه‌اندازی و استفاده به صورت آفلاین (بر روی سیستم شخصی خودتان)

### پیش‌نیازها
- 📦 نرم‌افزار XAMPP 8.2.12 یا بالاتر
- 🌐 مرورگر مدرن (Chrome, Firefox, Edge)
- 💻 سیستم عامل Windows, macOS یا Linux

### مراحل نصب

#### مرحله ۱: دانلود و نصب XAMPP
1. آخرین نسخه XAMPP را از [سایت رسمی](https://www.apachefriends.org/) دانلود کنید
2. نرم‌افزار را نصب کنید (پیشنهاد: مسیر پیش‌فرض `C:\xampp`)

#### مرحله ۲: کپی فایل‌های پروژه
```bash
# کپی محتویات پروژه به پوشه htdocs
C:\xampp\htdocs\
```

#### مرحله ۳: راه‌اندازی سرویس‌ها

1.XAMPP Control Panel را اجرا کنید<br>
2. سرویس‌های زیر را شروع کنید:<br>
    🟢 **Apache** - برای وب سرور<br>
    🟢 **MySQL** - برای پایگاه داده

#### مرحله ۴: ایجاد دیتابیس
1. در مرورگر به آدرس زیر بروید:
   ```
   http://localhost/phpmyadmin
   ```
2. یک دیتابیس جدید با نام `mdata` ایجاد کنید
3. از منوی Import، فایل `database/DatabaseMData.sql` را وارد کنید

#### مرحله ۵: اجرای برنامه
1. در مرورگر به آدرس زیر بروید:
   ```
   http://localhost/
   ```
2. با اطلاعات پیش‌فرض وارد سیستم شوید:
   - **👤 نام کاربری:** `admin`
   - **🔐 رمز عبور:** `admin123`
